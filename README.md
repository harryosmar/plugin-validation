# Validation
Composer plugin for validation purpose, contains set of validation rules.

## Installation

Add this composer.json file

```json
{
   "minimum-stability": "dev",
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:harryosmar/validation.git"
        }
    ],
    "require": {
        "harryosmar/validation": "dev-master"
    }
}
```

Then running
```bash
$ composer install
```
