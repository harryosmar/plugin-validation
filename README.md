## Installation

Add this composer.json file

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "git@github.com:harryosmar/plugin-validation.git"
        }
    ],
    "require": {
        "harryosmar/plugin-validation": "^2.0"
    }
}
```

Then running
```bash
$ composer install
```
